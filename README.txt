# MicBridge Pro - 安装说明

## 需要安装虚拟声卡

使用 MicBridge Pro 作为电脑麦克风，需要安装 **VB-Audio Virtual Cable**（免费软件）。

### 安装步骤

1. 下载 VB-Cable：
   - 官网：https://www.vb-audio.com/
     https://download.vb-audio.com/Download_CABLE/VBCABLE_Driver_Pack45.zip
   - 点击 "Virtual Cable" 下载 VBCable_Setup.exe

2. 运行 VBCABLE_Setup_x64.exe 安装

3. 重启电脑

4. 在 Windows 声音设置中：
   - **输入设备**：选择 "CABLE Input (VB-Audio Virtual Cable)"
   - **输出设备**：选择 "CABLE Output (VB-Audio Virtual Cable)"

### 快速测试

1. 打开 MicBridgePro.exe
2. 手机连接同一个 WiFi，打开 APP 选择 WiFi 模式，输入电脑 IP（已连接的WIFI属性-lpv4地址）
3. 在 QQ/微信 等软件的麦克风设置中选择 "CABLE Input"
